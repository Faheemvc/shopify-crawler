# ShopifyScrapper

This Web scrapper crawl through shopify inspiration websites and gives the products data as csv file 

## Installation
install required elixir

```elixir
def deps do
  [
    {:shopifyscrap, "~> 0.1.0"}
  ]
end
```
## Run following
```
mix deps.get
```
- Start server
```
mix phx.server
```
- Start server with iex
```
iex -S mix phx.server
```
